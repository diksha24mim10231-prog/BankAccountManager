# Bank Account Manager in Java

This is a simple console-based Bank Account Manager application developed in Java.  
It allows users to create bank accounts, deposit money, withdraw money, and view account details.

## Features

- Create new bank accounts with unique account numbers  
- View details of all accounts or a specific account  
- Deposit money into accounts  
- Withdraw money from accounts with balance checks  
- Simple text-based menu for easy operation

## Technology Used

- Java (no external libraries)  
- Console input/output  
- Object-Oriented Programming concepts

## How to Run

1. Open a terminal or command prompt.  
2. Navigate to the `src` folder inside the project directory.  
3. Compile the Java files with the command:  

javac BankAccount.java BankApp.java
4. Run the application with the command:  

java BankApp
